# Requirements

In order to run this application, you will need the following:

- [PHP](https://www.php.net/). Version 8.5 or greater is recommended.
- [Node](https://nodejs.org/en). Version 24 or higher is recommended.
- [Composer](https://getcomposer.org/). Version 2.9 or higher is recommended.

# Building and running the application
To build and run the application, follow these steps:

- Navigate to the base directory for this repository.
- Run `npm install && npm run build`.
- Run `composer run dev`.

It may take some time for the server to start. Once it's running, you can access the welcome page at `http://localhost:5173`.
